E5R Environment

Uma coleção de scripts para automação de tarefas de ambiente, para E5R Development Team.

E5R Environment, é uma coleção de scripts para automação de tarefas que dizem respeito ao ambiente de desenvolvimento do Time de Desenvolvimento E5R. Mas que pode facilmente ser usada por qualquer desenvolvedor (ou time de desenvolvimento) que pretenda aplicar os conceitos e padrões de desenvolvimento usados pelo E5R Development Team, em seus projetos.

https://github.com/e5r/env/blob/master/doc/ARCHITECTURE.md

Instalar

Instalar o ambiente de desenvolvimento E5R é tão simples quanto:

  1. Ctrl + C no código a seguir;
  2. Ctrl + V em um terminal;
  3. Enter e pronto!

~shell

e5r-install ... E5R Environment successfully installed!
