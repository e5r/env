E5R Environment

Uma coleção de scripts para automação de tarefas de ambiente, para E5R Development Team.

Notícias

  • Lançamento da versão 0.1.0-alpha1

Instalar

  1. Baixe e execute o script e5r-install de acordo com o sistema operacional
    • Windows
    • Unix (em breve)
  2. A mensagem E5R Environment successfully installed! é apresentada

    e5r-install

    'E5R Environment successfully installed!'
